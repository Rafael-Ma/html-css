Usando Links

Links externos

Podemos ancorar links utilizando a tag A.

Exemplo:


    <p>Você pode acessar o <a href="https://gustavoguanabara.github.io">Repositório do Guaba</a>, no GitHub.<p>

Você pode acessar o Repositório do Guaba, no GitHub.

O ideal é que seja utilizado os seguintes parâmetros:


    <p>Você pode acessar o <a href="https://gustavoguanabara.github.io" target="_blank" rel="external">Repositório do Guaba</a>, no GitHub.</p>

Você pode acessar o Repositório do Guaba, no GitHub.

Links internos

Podemos ancorar documentos de páginas usando a tag A (o parâmetrol REL identifica se a página será a seguinte ou enterior).


    <p>Esta é a primeira página do site. Se você quiser, pode acessar também a minha <a href="pag002.html" rel="next" target="_self">segunda página</a>.</p>

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página.

Você também pode acessar a nossa página de notícias
