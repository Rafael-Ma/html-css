Trabalhando com listas

Listas Ordenadas

Usamos a tag OL para criar uma lista, e a tag LI para adicionarmos um item (/LI não é necessário para o fechamento da tag.).


    <ol>
        <li>Acordar</li>
        <li>Tomar café</li>
        <li>Escovar os dentes</li>
    </ol>
  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Atualiza numeração automaticamente ao adicinarmos mais itens.

Podemos usar o parâmetro TYPE para mudar o formato da lista.


    <ol type="A">
        <li>Acordar</li>
        <li>Tomar café</li>
        <li>Escovar os dentes</li>
    </ol>
  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Tipos:

  1. "1" = Numérico
  2. "A" = Alfabeto maiúsculo
  3. "a" = Alfabeto minúsculo
  4. "I" = Romano maiúsculo
  5. "i" = Romano minúsculo

Listas não Ordenadas

Usamos a tag UL.


    <ul>
        <li>Pão</li>
        <li>Leite</li>
        <li>Tomate</li>
        <li>Manteiga</li>
        <li>Arroz</li>
        <li>Feijão</li>
    </ul>
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Também podemos usar o parâmetro TYPE


    <ul type="circle">
        <li>Pão</li>
        <li>Leite</li>
        <li>Tomate</li>
        <li>Manteiga</li>
        <li>Arroz</li>
        <li>Feijão</li>
    </ul>
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Tipos:

  • "disc" = Círculo preenchido
  • "circle" = Círculo vazio
  • "square = Quadrado

Listas Mistas

Podemos colocar listas dentro de outras


    <ol>
        <li>NES</li>
        <ul>
            <li>Mario Bros</li>
            <ul type="circle">
                <li>Mario Bros 3</li>
                <li>Mario: Lost Levels</li>
            </ul>
            <li>Ninja Gaiden</li>
        </ul>
        <li>SNES</li>
        <ul>
            <li>Mario</li>
            <li>Donkey Kong</li>
        </ul>
        <li>PlayStation</li>
        <ul>
            <li>Final Fantasy</li>
            <li>Castlevania</li>
        </ul>
    </ol>

Lista de Definições

Usamos a tag DL. Com as tag DT para o termo e DD para a definição.


    <dl>
        <dt>Termo</dt>
        <dd>Definição</dd>
        <dt>Segundo Termo</dt>
        <dd>Definição do Segundo Termo</dd>
    </dl>

Lista:

Termo
Definição
Segudo Termo
Definição do Segundo Termo
