Uso de Grouping tags

Podemos usar várias tags para organizar o conteúdo de nosso site, sem termos que usar apenas div, deixando o site com melhor engajamento semântico, como as tags header main footer para organizar o cabeçalho, conteúdo principal e rodapé respectivamente.

Também podemos utilizar a tag nav para criarmos menus de navegação, geralmente dentro do cabeçalho (header).

A tag section é usada para determinar as sessões ou partes de nosso conteúdo, como por exemplo: <section id="noticias">.

Como o própio nome sugere, a tag article é utilizada para escrever artigos dentro de nosso conteúdo. Ótimo para delimitarmos no serviço de busca dos navegadores de que se trata de um artigo.

A tag aside é utilizada para apresentar informações tangenciais ao conteúdo principal, exemplo:

<aside>
    <p>Artigo escrito por Rafael Matiola</p>
</aside>
